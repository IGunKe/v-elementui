### git 使用

git status 查看当前修改信息
git add . 添加到暂存区
git commit -m "消息" 提交到本地仓库
git branch 查看分支

合并分支要先切换到指定分支
如 git checkout master
git marge 分支名


git push -u origin login 把本地分支推送到远程仓库
